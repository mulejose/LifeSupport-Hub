# LifeSupport Hub 🌟

A modern, motivational website designed to help people overcome life's challenges with resources, community support, and interactive wellness tools.

## 🎯 Features

### 🌱 **Inspirational Content**
- Daily motivational quotes to inspire and uplift
- Affirmations for daily motivation
- Real success stories from community members

### 📚 **Comprehensive Resources**
- **Mental Health**: Guides on anxiety, depression, and stress management
- **Career Support**: Job transition help and professional development
- **Financial Wellness**: Budget planning and debt management
- **Physical Health**: Fitness routines and nutrition tips
- **Relationships**: Communication and connection building
- **Personal Growth**: Self-discovery and habit development

### 🛠️ **Interactive Tools**
- **Daily Affirmation Generator**: Get personalized affirmations
- **Breathing Exercise Guide**: Interactive breathing techniques
- **Progress Tracker**: Set and track weekly goals (saved locally)

### 👥 **Community Features**
- Connect with 50K+ active members
- Share experiences and celebrate wins
- Weekly challenges and mentorship opportunities
- Safe, supportive environment for growth

### 📱 **Modern & Responsive Design**
- Clean, professional interface
- Fully responsive (works on mobile, tablet, desktop)
- Smooth animations and transitions
- Accessibility-first approach

## 🚀 Quick Start

### Option 1: View Online
1. Go to your GitHub repository: `https://github.com/mulejose/LifeSupport-Hub`
2. Click "Settings" → "Pages"
3. Set source to "Deploy from a branch" → "main" → "/ (root)"
4. Your site will be live at `https://mulejose.github.io/LifeSupport-Hub/`

### Option 2: Local Development
```bash
# Clone the repository
git clone https://github.com/mulejose/LifeSupport-Hub.git
cd LifeSupport-Hub

# Open in your browser (use a local server for best results)
# Using Python 3:
python -m http.server 8000

# Using Python 2:
python -m SimpleHTTPServer 8000

# Then visit: http://localhost:8000
```

## 🚀 Getting Started for Developers

### Setup
```bash

```

### Customization

**Change Colors**: Edit variables in `styles.css`
```css
:root {
    --primary-color: #7b68ee;
    --accent-color: #20c997;
    /* ... other colors */
}
```

**Add New Quotes**: Edit `quotes` array in `script.js`
```javascript
const quotes = [
    { text: "Your quote here", author: "Author" },
    // Add more quotes
];
```

**Customize Content**: Edit HTML in `index.html`
