## 🚀 **DEPLOY YOUR WEBSITE (3 SIMPLE STEPS)**

### Step 1: Go to Repository Settings
```

```

### Step 2: Configure GitHub Pages
- **Source**: Select "Deploy from a branch"
- **Branch**: Select "main"
- **Folder**: Select "/ (root)"
- Click **Save**

Wait 1-2 minutes, then visit:
